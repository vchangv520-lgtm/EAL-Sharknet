## 1. 怎么放映

纯静态网页，不需要构建。

```bash
cd /Applications/Documents/dash/shark-net-presentation
python3 -m http.server 4178
# 浏览器打开 http://localhost:4178   （建议按 F 全屏 / 或浏览器全屏放映）
```

- 翻页：`← →` `↑ ↓` `空格` `PageUp/Dn`；`Home/End` 跳首尾；也可点右下角箭头、手机左右滑。
- 地址栏加 `#5` 可直接跳到第 5 张（方便排练某一段）。
- 配合演讲：**一张 = 一个论点节拍**，讲到哪点按一下翻到对应页即可。

> ⏱ 这是为 5 分钟口头演讲（EAL Outcome，截止 2026-06-22）做的背景大屏，
> 横屏放映效果最佳；窄屏（手机）会自动改为可滚动布局。

## 4. 想改的话 —— 三件需要知道的事

### (a) 改文字 / 数字
直接在 `index.html` 对应 `<section>` 里改。**每张页上方都有一段 HTML 注释**
说明这张在讲什么、动画怎么设计的。
⚠ 数据务必与第 6 节的演讲原文一致（223 / 24 / 3400+ / 676 / 284 / 190 / 133 /
1937 / 2025，以及来源 ABC / SBS / The Guardian），别改错。

### (b) 改颜色 / 字体 / 间距
只动 `styles.css` 顶部的 `:root` token。颜色一律用变量（如 `var(--teal)`），
不要写死色值。字体只有两种：标题 `Space Grotesk`、正文 `Inter`。

颜色语义（全片视觉逻辑，改时保持一致）：
- **青绿 / lime（`--teal` `--cyan` `--lime`）= 智能、无害、海洋、我方主张**
- **珊瑚红（`--coral`）= 鲨鱼网、致命、反方/旧方法**
- **琥珀黄（`--amber`）= 关键统计数字高亮**

### (c) 加动画 —— 不用写 JS，只加 4 个 data-属性

| 属性 | 作用 | 例子 |
|------|------|------|
| `data-reveal` | 元素在本页打开时淡入上移 | `<h2 class="h2" data-reveal>` |
| `data-delay="240"` | 自定义错峰延迟（毫秒） | `data-reveal data-delay="240"` |
| `data-count="676"` | 数字从 0 滚到目标（配 `data-suffix` / `data-prefix` / `data-dec` / `data-dur`） | `<span data-count="3400" data-suffix="+">0</span>` |
| `data-grow="42%"` | 柱子/进度条长到指定高/宽 | `<div class="bar bar--pro" data-grow="42%">` |

补充变体：`.from-right`（改成从右滑入）、`.pop`（缩放弹入）。
waffle 网格写 `<div class="waffle" data-total="100" data-fill="11">`，格子由引擎自动生成。

### 可复用组件速查

## 6. 演讲全文（事实核对基准，勿改数据）

- 主张：澳洲各州应**淘汰鲨鱼网**，改用无人机、SMART drumline、监听站、实时警报。
- 鲨鱼网只是离岸的一段网，**不是墙**；鲨鱼可从上/下/侧绕过。NSW 自 **1937** 年使用。
- **2025 年 9 月**，Dee Why 在已装网情况下仍发生**致命**袭击（来源：SBS）。
- NSW **2024–25** 赛季网中捕获 **223** 只动物，仅 **24** 只为目标鲨；昆州一年内网与 drumline
  害死 **3,400+** 只鲨（来源：ABC）。误捕含海豚、鲸、海龟、儒艮、灰护士鲨、双髻鲨、鳐（来源：The Guardian / ABC）。
- 昆州无人机试验：无人机 **676** 次目击 vs 网与 drumline 捕获 **284**；
  >2 米的鲨，无人机看到 **190** 条 vs 网/drumline **133** 条（来源：The Guardian / ABC）。
- SMART drumline：钩到即报警 → 标记放生；监听站 + App 在标记鲨靠近时警示泳客。
- 结论：鲨鱼网在 1930 年代或许合理，但 2026 年我们有更好选择 —— 既保护人，也保护海洋。
